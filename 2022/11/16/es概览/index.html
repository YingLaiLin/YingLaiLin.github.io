ES 概览

基本概念

ElasticSearch是一款非常强大的、基于 Lucene 的开源搜索及分析引擎, 可以用于构建近实时的分布式搜索分析引擎.

其内部通过使用 RESTful API, 以隐藏 Lucene 的复杂性.

与关系型数据库的概念对比
| 关系型数据库 | ES |
| ———– | ———– |
| 数据库 | 索引(index) |
| 表 | 类型(type, 6.0.0 废弃) |
| 行 | 文档(document) |
| 列 | 字段(field) |
| 表结构 | 映射(mapping) |
| 索引 | 反向索引(不确定?) |

使用场景

  • 搜索类场景
    • 网站搜索
    • 垂直搜索
    • 代码搜索

分析类场景
- 日志管理与分析
- 安全指标监控
- Web抓取舆情分析
